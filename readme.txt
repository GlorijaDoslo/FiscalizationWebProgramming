You need to install this too(frontend):

ng add @angular/material

To install ngx-pagination enter this command in terminal (location while executing this command should be before frontend and backend folders):

npm install ngx-pagination --save(http://michaelbromley.github.io/ngx-pagination/#/)

To install ngx-charts enter the following command in terminal(in frontend):

npm i @swimlane/ngx-charts --save


Additional command:

npm install @material/dialog
